# PlantCare AI 🌱

PlantCare AI is a real-time plant disease detection system built using Python, OpenCV, and TensorFlow Lite.

## Features
- Real-time camera-based plant monitoring
- Plant disease classification
- TensorFlow Lite optimized model
- Lightweight and fast inference

## Technologies Used
- Python
- OpenCV
- TensorFlow Lite
- NumPy

## Project Structure

PlantCare_AI/
├── main.py
├── camera_test.py
├── converted_tflite_quantized/
│ ├── model.tflite
│ └── labels.txt

## How to Run

1. Install dependencies
2. Connect webcam
3. Run:

```bash
python main.py
```
